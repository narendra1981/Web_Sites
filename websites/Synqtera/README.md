# SynQtera Technologies Website

A modern static website built with HTML, CSS, and JavaScript.

## Project Structure

```
Synqtera/
├── index.html          # Main HTML file
├── css/
│   └── style.css      # Stylesheet
├── js/
│   └── main.js        # JavaScript functionality
├── img/               # Image assets folder
└── README.md          # This file
```

## Features

- **Responsive Design**: Mobile-friendly layout that adapts to all screen sizes
- **Modern Styling**: Clean and professional CSS with gradients and animations
- **Smooth Navigation**: Smooth scrolling between sections
- **Service Cards**: Showcasing your services or products
- **Contact Section**: Ready for you to add contact information

## Getting Started

1. Open `index.html` in your web browser
2. Customize the content to fit your needs
3. Add your images to the `img/` folder
4. Update the CSS in `css/style.css` for your branding
5. Add more functionality in `js/main.js`

## Customization

- **Colors**: Edit the color values in `css/style.css` (look for hex codes like `#667eea`)
- **Content**: Update text in `index.html`
- **Logo**: Replace "Synqtera" with your brand name
- **Navigation**: Add or remove menu items in the `<nav>` section

## Sections Included

- **Header/Navigation**: Sticky navigation bar
- **Hero Section**: Eye-catching introduction
- **About**: Brief description of your business
- **Services**: Display your services or products
- **Contact**: Call-to-action section
- **Footer**: Copyright information

## Browser Support

Works on all modern browsers including:
- Chrome
- Firefox
- Safari
- Edge

## Development Tips

- Use the browser's Developer Tools (F12) to test responsive design
- Test navigation and button clicks
- Validate HTML/CSS using W3C validators
- Optimize images before adding to the `img/` folder
